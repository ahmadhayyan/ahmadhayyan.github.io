Service Report is a desktop application to record employee activities in servicing the company's hospital laboratory equipment which is leased to customers. The application can record which company the device is being rented on, the date the error was recorded, the error, how to handle the error and also who the technician repaired it.

Features :
- Login system
- Multiple user groups with their own accessibility
- Track hospital laboratory equipment service report
- Print hospital laboratory equipment service report
- Upload a pictures (pictures of hospital laboratory equipment malfunction)

Benefits :
- Minimizing misinformation
- Able to track the status of hospital laboratory equipment
