Timer Socket (T-SOCK) is a device with a timer function built into a socket where the socket itself will turn on and off continuously according to the user-defined length of on and off times. T-SOCK is an arrangement of several components that make up the T-SOCK working system itself. The work system is divided into 3 parts, namely relays, menu buttons, and Arduino Nano.

Features :
- Able to set the T-Sock timer to run only 1 time
- Able to set how long the timer is on and off
- Able to set the T-Sock timer to run continuously switching between on and off

Benefits :
- Become a device that can solve problems faced when forgetting to unplug an electronic device connected to electricity
- Become a device that can help save daily electricity usage
