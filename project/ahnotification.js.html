ahNotification.js is a javascript library that provides simple, lightweight, stylish, responsive, and functional/interactable notifications.

Documentation and demos at ahnotification.js.org
Or view it on GitHub.

Features :
- 4 types of flash messages
- Custom action buttons
- Support CSS & HTML tag

Benefits :
- The user becomes more aware of whether the action was successful or failed
- Custom action buttons can be used for web applications
