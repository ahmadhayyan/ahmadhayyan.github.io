Web Store is a website application for buying and selling items and Admin Panel is a website application for managing order payments and serial numbers.

Web Store Features :
- Responsive design
- Automatically increase or clear payment id
- Multiple payment method
- Buyer gets a receipt in the email
- Admin gets an email notification if there is an incoming order

Admin Panel Features :
- Responsive design
- Login system
- Confirm or decline order payment
- Automatically search for an available serial number when confirming payment
- Automatically email buyers after confirming or declining payment
- Automatically increases revenue when confirming payment
- Manage serial numbers
- Able to generate serial number automatically

Benefits :
- Can buy items easily through the web store
- Can easily manage order payments and serial numbers using just a cell phone, laptop, or computer.
