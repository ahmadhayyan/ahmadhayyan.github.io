Face Detection System for Advertising Placement is a device that can count the human faces viewing an ad to provide best position statistics for ad placement without a market density survey. This device uses a face detection , face tracking and face counting systems. After a new face is counted, the system will save the data to a CSV file, then the CSV file will be displayed on the website.

Benefit :
- Minimizing errors in ad placement
- Advertisers get more viewers so that they are more widely known

Web Design Output :

A Page that contains a graph and a time system capturing human faces.

Service Report is a desktop application to record employee activities in servicing the company's hospital laboratory equipment which is leased to customers. The application can record which company the device is being rented on, the date the error was recorded, the error, how to handle the error and also who the technician repaired it.

Feature :
- Login system
- Track hospital laboratory equipment service report
- Print hospital laboratory equipment service report
- Upload image ( the damage )

SMS Storage is a desktop application for managing company inventory to control so that the first item entered comes out first, thereby minimizing expired items. The application is connected to the office database used for invoice creation, so that the application can read invoice items which make warehouse employees prepare items faster than waiting for the admin to give several invoices to warehouse employees and prepare them at once. This application is also able to print invoice items with the serial number selected by the system which will be used by warehouse employees to prepare items.

Feature :
- Login system
- Manage office inventory ( date of entry, item type, item name, serial number, quantity, expired date, status )
- Read invoices created by admin from the accounting app
- Print invoices items

Fogponic Plant Grower Box is a plant growing tool that uses the fogponic growing method, where the fogger will convert the nutrient and water solutions into particles measuring 5 - 30 µm to be sprayed on plants, and can work automatically by relying on the Arduino nano microcontroller. This tool has a 10W LED light that can be adjusted to the brightness of the red and blue colors to resemble the sun so that the tool is suitable when used indoors. Another capability of this tool is that it can monitor the humidity of the tool, the water temperature, and the water level limit for the fogger. The data will be displayed on an LCD and the tool will automatically adjust the situation to match the ideal conditions needed by the plant, for example the tool will turn on the fogger when the humidity of the tool is below the specified limit and turn off the fogger when the humidity of the tool is above the specified limit.

Benefit :
- Solve the problem of shrinking agricultural land in Indonesia
- Reduce cases of chemical poisoning in food, so that it is useful to help people maintain health quality

Timer Socket (T-SOCK) is a device with a timer function built in to a socket where the socket itself will turns on and off continuously according to the user-defined length of ON and OFF time. T-SOCK is an arrangement of several components that make up the T-SOCK working system itself. The work system is divided into 3 parts, namely relays, menu buttons, and Arduino nano.

Benefit :
- Become a device that can solve problems faced when forgetting to unplug an electronic device connected to electricity
- Become a device that can help save daily electricity usage
